L (エル, Eru) este un detectiv de renume mondiala care își asumă provocarea de a prinde criminalul în masă cunoscut sub numele de Kira. În ancheta sa, L devine suspicios față de Light Yagami și își propune să demonstreze că Light este Kira.

L este foarte inteligent, deși aspectul său dezordonat și languid îi maschează marile puteri de deducere și mulți îi pun la îndoială abilitățile la vizionarea lui, L este extrem de meticulos și analitic. Se pricepe să-și înșele adversarii și este dispus să ia măsuri drastice pentru a rezolva un caz.

Deși este slab, el mănâncă numai dulciuri, și este foarte sedentar. El are și tendinta de-a se pune cu picioarele pe scaun când sade.
